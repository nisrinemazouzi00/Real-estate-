# 🏠 GP1 – Real Estate Web Application

## 📌 Overview
GP1 is a full-stack real estate web application built using Node.js for the backend and HTML, CSS, and JavaScript for the frontend.

The application allows users to:
- Browse property listings
- Search for properties
- Register and login
- View detailed property pages
- Send contact inquiries

---
